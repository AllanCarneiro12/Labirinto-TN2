# Labirinto TN2

## Descrição
Projeto desenvolvido em linguagem C para a disciplina de Algoritmos II, ministrada pelo Professor Alessandro Brassanini.

Este é um jogo de labirinto com armadilhas (bombas), sistema de pontuação de desempenho e efeitos visuais e sonoros. O objetivo é chegar ao fim do labirinto com o menor número de passos possível, evitando cair nos buracos deixados pelas bombas.

## Autores
- Allan Carneiro da Cunha Silveira
- Isadora Voss Brugnera
- Jenifer Beatriz Nunes Ribeiro

## Como Jogar
- Navegue pelo labirinto usando as teclas de direção
- Evite as bombas que deixam buracos pelo caminho
- Tente completar o labirinto com o menor número de passos
- Sua pontuação é baseada no desempenho ao longo do percurso

## Compilação
```bash
gcc Labirinto.c -o Labirinto
```

## Execução
```bash
./Labirinto
```
